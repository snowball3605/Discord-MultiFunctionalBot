# 什麼是 MultiFunctionalBot(MFB) ?
MFB 是一個多功能,多語言,權限設定的機器人.
    
# Discord.java V19 MultiFunctionalBot - 介紹:
你可以查看語言代碼和使用指令自由互換語言.最重要的是它還可以指定把一些你不想用的指令關閉<br>
    
如果您喜歡這隻機器人話，您可以為我們點一下 (⭐️) 來支持一下我們這個機器人!
